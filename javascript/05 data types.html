Data Types

Javapre variables can hold many data types: numbers, strings, objects and more. Javapre Types are Dynamic. This means that the same variable can be used to hold different data types: 
var x;         // Now x is undefined
x = 5;         // Now x is a Number
x = "John";      // Now x is a String
document.write( x );

Strings

A string (or a text string) is a series of characters. Strings are written with quotes. You can use single or double quotes. 
var answer1 = "It's alright";
var answer2 = "He is called 'Johnny'";
var answer3 = 'He is called "Johnny"';
document.getElementById("demo").innerHTML =
answer1 + "<br>" + 
answer2 + "<br>" + 
answer3;

Numbers

Javapre has only one type of numbers. Numbers can be written with, or without decimals. 
var y = 123;
var z = 123e-5;
document.write(y + "<br>" + z);

Booleans

Booleans can only have two values: true or false. 
var x = 5;
var y = 7;
document.write(x > y);

Arrays

Javapre arrays are written with square brackets and array items are separated by commas. 
var cars = ["Saab","Volvo","BMW"];
document.write(cars[0]);

Adding Numbers and Strings

Javapre uses the + operator for both addition and concatenation. Numbers are added. Strings are concatenated. If you add two numbers, the result will be a number: 
var x = 10;
var y = 20;
var z = x + y;
If you add two strings, the result will be a string concatenation: 
var x = "10";
var y = "20";
var z = x + y;
If you add a number and a string, the result will be a string concatenation: 
var x = 10;
var y = "20";
var z = x + y;

JSON

  • JSON is a format for storing and transporting data. JSON is often used when data is sent from a server to a web page.
  • JSON stands for JavaScript Object Notation
  • JSON is a lightweight data interchange format
  • JSON is language independent *
  • JSON is "self-describing" and easy to understand
  • JSON Syntax Rules
  • Data is in name/value pairs
  • Data is separated by commas
  • Curly braces hold objects
  • Square brackets hold arrays

JavaScript Objects

JavaScript objects are written with curly braces {}. Object properties are written as name:value pairs, separated by commas. 
var person = {
  firstName : "John",
  lastName  : "Doe",
  age     : 50,
  eyeColor  : "blue"
};
document.write(person.firstName + " is " + person.age + " years old.");
