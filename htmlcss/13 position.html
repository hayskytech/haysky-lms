Position

The position property specifies the type of positioning method used for an element (static, relative, fixed or absolute). position: static; An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page. 
div.static {
  position: static;
  border: 3px solid #73AD21;
}
position: relative; An element with position: relative; is positioned relative to its normal position. 
div.relative {
  position: relative;
  left: 30px;
  border: 3px solid #73AD21;
}
position: fixed; An element with position: fixed; is positioned relative to the viewport. 
div.fixed {
  position: fixed;
  bottom: 0;
  right: 0;
  width: 300px;
  border: 3px solid #73AD21;
}
position: absolute; An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed). 
div.relative {
  position: relative;
  width: 400px;
  height: 200px;
  border: 3px solid #73AD21;
}
b.absolute {
  position: absolute;
  top: 80px;
  right: 0;
  width: 200px;
  height: 100px;
  border: 3px solid #73AD21;
}
