React JS

Installation

Install in current folder

npx create-react-app .

or Install in sub folder (practice_app)

npx create-react-app practice_app

Run / Start server

npm start

visit localhost:3000

Bootstrap

npm install react-bootstrap bootstrap

The following line can be included in your src/index.js or App.js file

import 'bootstrap/dist/css/bootstrap.min.css'

Semantic

npm install semantic-ui-react semantic-ui-css

The following line can be included in your src/index.js or App.js file

import 'semantic-ui-css/semantic.min.css'

Elements and Attributes

After adding components. Replace the following:

class        	=>          className
href="#"      =>          href="/"
<hr>          =>          <hr/>

Components

Create a component

In src folder - create new folder components and create new js file MyComponent.js - press (rfc) & enter

VSCode Extension: ES7 React/Redux/GraphQL/React-Native snippets

press rfc to get the below code:

export default function MyComponent() {
	return (
		<div>Hello</div>
	)
}

In src/App.js add this to the top

import MyComponent from './components/MyComponent';

In src/App.js add this in html part

<MyComponent/>

More than one functional components in one file

export const ComponentOne = () => <span>Some content for ComponentOne</span>;
export const ComponentTwo = () => <span>Some content for ComponentOne</span>;
export function About() {
  return (
    <span>Some content for About</span>
  )
}

{ } needed because it is not exported by default

import { ComponentOne, ComponentTwo, Html } from "./path/of/components"

Styles

import React from 'react'
function Student() {
	const student = {
		border: '1px solid black', 
		padding: '10px', 
		margin: '10px', 
		display: "inline-block" 
	}
	return (
		<div style={student}>
			<h2>Student Name
			<h3>Town: NDK
			<h3>Phone: 112233
		</div>
	)
}	
export default Student
